# RUANG KERJA GURU HARIAN

## Nama Menu

📚 Mengajar Hari Ini

Menu ini bukan hanya untuk melihat materi, tetapi menjadi pusat kerja guru sejak masuk kelas sampai selesai KBM.

---

## Alur Guru di Kelas

1. Guru masuk kelas.
2. Guru membuka menu Mengajar Hari Ini.
3. Guru melakukan absen siswa.
4. Guru mengecek urusan keuangan siswa:
   - iuran,
   - tabungan,
   - tunggakan,
   - catatan pembayaran.
5. Guru mengecek notifikasi:
   - PR yang harus diperiksa,
   - tugas yang belum dinilai,
   - siswa yang belum mengumpulkan.
6. Guru mulai mengajar sesuai jadwal.
7. Guru menutup KBM dengan status:
   - Terlaksana,
   - Terlaksana Sebagian,
   - Belum Terlaksana.
8. Guru dapat menambahkan:
   - PR baru,
   - tugas baru,
   - nilai,
   - kokurikuler,
   - catatan guru.
9. Aplikasi menyimpan semuanya menjadi jurnal otomatis.

---

## Urutan Tampilan di Aplikasi

### 1. Kartu Pembuka

Berisi:
- nama guru,
- tanggal,
- kelas,
- jadwal hari ini.

### 2. Absen Siswa

Guru menandai:
- hadir,
- sakit,
- izin,
- alfa.

### 3. Keuangan Siswa

Untuk Guru Bendahara:
- input iuran,
- input tabungan,
- lihat tunggakan.

Untuk Guru Murni:
- hanya melihat ringkasan jika diperlukan,
- tidak ada input bendahara.

### 4. Notifikasi PR dan Tugas

Menampilkan:
- PR yang harus diperiksa hari ini,
- tugas yang belum dinilai,
- siswa yang belum mengumpulkan.

### 5. Materi Mengajar

Berisi:
- mapel,
- jam,
- TP,
- materi,
- strategi pembelajaran,
- media,
- asesmen.

### 6. Penutup KBM

Guru memilih:
- Terlaksana,
- Sebagian,
- Belum terlaksana.

Jika sebagian atau belum terlaksana, aplikasi membuat jadwal ulang otomatis.

### 7. Tambahan Aktivitas

Guru bisa mengisi:
- PR baru,
- tugas baru,
- nilai,
- kokurikuler temporer,
- catatan guru.

### 8. Simpan Jurnal

Semua data otomatis masuk ke:
- jurnal harian guru,
- rekap KBM,
- rekap PR,
- rekap tugas,
- rekap kokurikuler,
- bahan evaluasi,
- bahan akreditasi.
