# STANDAR UI SIPINTAR_MI v10

## 1. Prinsip Utama

Tampilan SIPINTAR_MI harus:

- ramah HP,
- artistik,
- berwarna,
- tidak membosankan,
- mudah dipakai guru,
- tombol besar,
- minim tabel panjang,
- terasa seperti aplikasi Android modern.

---

## 2. Konsep Login

Login hanya menampilkan:

- logo/nama aplikasi,
- nama madrasah,
- username,
- password,
- tombol masuk.

Background:
- artistik,
- nuansa madrasah,
- gunung/kabut/hijau lembut.

---

## 3. Dashboard Guru

Menu utama guru dibuat dengan ubin besar.

Ubin utama paling besar:

- 📚 Mengajar Hari Ini

Ubin tambahan:

- 📖 Perangkat
- 📅 Kalender
- 📊 Laporan Saya

Untuk Guru Bendahara ditambah:

- 💰 Iuran
- 🏦 Tabungan
- 💵 Bendahara

---

## 4. Warna Menu

- Mengajar Hari Ini: biru-hijau
- Perangkat: biru
- Kalender: hijau
- Laporan: ungu
- Absensi: hijau muda
- Keuangan: ungu
- PR/Tugas: oranye
- Kokurikuler: toska
- Notifikasi: kuning
- Admin: abu-abu

---

## 5. Ruang Kerja Guru Harian

Halaman Mengajar Hari Ini dibuat seperti timeline:

1. Absen
2. Keuangan siswa
3. Notifikasi PR/Tugas
4. Materi mengajar
5. Penilaian
6. PR/Tugas baru
7. Kokurikuler
8. Simpan jurnal

Setiap bagian diberi warna berbeda.

---

## 6. Progress Harian

Di bagian atas halaman Mengajar Hari Ini ada progress:

- Absen
- Keuangan
- Cek PR
- Mengajar
- Penilaian
- Jurnal

Contoh status:

- selesai
- sedang berjalan
- belum dikerjakan

---

## 7. Tampilan Mobile

Aturan mobile:

- ubin dashboard maksimal 2 kolom,
- tombol full width,
- form satu kolom,
- tabel bisa scroll horizontal,
- teks cukup besar,
- jarak antar elemen lega,
- sidebar tidak menjadi pusat navigasi.

---

## 8. Nuansa Bahasa

Bahasa aplikasi harus:

- sopan,
- ringan,
- tidak kaku,
- cocok untuk guru madrasah.

Contoh sapaan:

"Assalamu'alaikum, Pak/Bu Guru. Semoga Allah mudahkan amanah mengajar hari ini."
